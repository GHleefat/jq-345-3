## 1. 产品概述

「墨韵」是一款基于 Canvas 的网页水墨画创作工具，模拟真实水墨在宣纸上的扩散、晕染效果。用户可使用毛笔书写运笔、清水晕开墨迹、墨滴入水扩散花纹，在数字画布上体验传统水墨写意画的创作乐趣。
- 目标用户：书法爱好者、水墨画创作者、寻求放松体验的用户
- 核心价值：将传统水墨画的沉浸式体验带入数字世界，零门槛创作写意画

## 2. 核心功能

### 2.1 功能模块
1. **画布页面**：全屏水墨画创作区、工具栏、设置面板
2. **笔画记录**：记录每一笔的时间、笔刷、坐标和参数
3. **创作回放**：原速或倍速回放整个创作过程，支持进度条控制
4. **视频导出**：将回放导出为视频或动图，支持分享

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 画布页面 | 创作画布 | 全屏Canvas画布，支持鼠标/触控绘画，模拟宣纸纹理 |
| 画布页面 | 毛笔工具 | 根据运笔速度和压力改变笔触粗细和墨色浓淡，笔触边缘有晕染扩散效果 |
| 画布页面 | 清水工具 | 拖动清水笔触可晕开已有墨迹，产生水墨交融的渲染效果 |
| 画布页面 | 墨滴工具 | 点击画面滴入墨滴，墨滴在宣纸上向四周随机扩散，产生不规则墨花 |
| 画布页面 | 墨色控制 | 调节墨色浓淡（淡墨、浓墨、焦墨），墨色随停留时间加深 |
| 画布页面 | 宣纸纹理 | 可切换不同宣纸纹理背景（生宣、熟宣、皮纸） |
| 画布页面 | 导出功能 | 将作品导出为高清PNG图片 |
| 画布页面 | 撤销/清空 | 支持撤销操作和清空画布 |
| 画布页面 | 笔画录制 | 开始/停止录制创作过程，记录每一笔的时间、笔刷类型、坐标、压力、墨色等参数 |
| 画布页面 | 创作回放 | 支持原速或倍速（0.5x-8x）回放，可拖动进度条跳转到任意时刻 |
| 画布页面 | 视频导出 | 将创作过程导出为WebM视频或GIF动图，可调节帧率、画质和播放速度 |
| 画布页面 | 分享功能 | 支持导出的视频可直接分享到微信朋友圈等社交平台 |

## 3.1 笔画录制流程

用户点击录制按钮 → 画布清空 → 开始记录 → 创作过程中每一笔都被记录（时间戳、坐标、压力、笔刷参数）→ 点击停止录制 → 生成录制会话 → 显示回放控制条

## 3.2 创作回放流程

用户有录制会话 → 点击回放按钮 → 进入回放模式 → 显示回放控制面板 → 播放创作过程 → 可随时暂停/继续 → 可调节播放速度（0.5x-8x）→ 可拖动进度条跳转 → 可导出为视频

## 3.3 视频导出流程

在回放模式下 → 点击导出按钮 → 弹出导出面板 → 选择格式（视频/GIF）→ 调节参数（帧率、画质、速度）→ 点击导出 → 逐帧渲染 → 自动下载文件 → 可分享到社交平台

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：水墨黑(#1a1a1a) + 宣纸白(#f5f0e8) + 朱砂红点缀(#c23b22)
- **辅助色**：淡墨灰(#666)、浓墨色(#2a2a2a)、水印蓝(#4a7c8a)
- **按钮风格**：圆角药丸形，带有水墨晕染边缘效果
- **字体**：展示字体用"站酷仓耳渔阳体"或"Ma Shan Zheng"（Google Fonts中文书法字体），UI字体用"Noto Serif SC"
- **布局风格**：全屏画布 + 浮动工具栏，沉浸式创作体验
- **图标风格**：线描风格中国风图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 画布页面 | 创作画布 | 全屏Canvas，宣纸纹理底色，边角微卷效果 |
| 画布页面 | 浮动工具栏 | 左侧垂直浮动栏，毛笔/清水/墨滴图标按钮，朱砂红高亮选中 |
| 画布页面 | 墨色滑块 | 工具栏下方，淡墨→浓墨渐变滑块 |
| 画布页面 | 笔触大小 | 工具栏下方，圆形大小预览+滑块调节 |
| 画布页面 | 右上角操作栏 | 撤销、清空、宣纸切换、录制、回放、导出按钮，半透明背景 |
| 画布页面 | 宣纸切换面板 | 弹出式面板，展示三种宣纸纹理缩略图 |
| 画布页面 | 录制指示 | 顶部中央红色脉冲动画指示，显示"录制中" |
| 画布页面 | 回放控制面板 | 底部中央浮动面板，包含播放/暂停、进度条、倍速选择、笔画计数、时间显示 |
| 画布页面 | 导出面板 | 右下角弹出面板，包含视频/GIF格式选择、帧率、画质、速度调节 |

### 4.3 响应式设计
- 桌面优先设计，支持鼠标和触控板操作
- 平板适配：支持触控绘画和压力感应
- 移动端：工具栏折叠为底部浮动栏，回放控制器适配屏幕宽度

### 4.4 交互细节
- 毛笔绘画时，笔触实时渲染带扩散效果，视觉上模拟墨在宣纸上的洇开
- 清水工具拖过墨迹时，墨迹边缘向四周自然晕散
- 墨滴点击后，墨花以动画形式逐渐扩散开来
- 工具切换时带有水墨转场微动画
- 导出按钮点击后有印章盖下的动画反馈
- 录制按钮点击后有红色脉冲动画，画布顶部显示录制状态
- 回放时画布隐藏工具栏和操作栏，专注于回放内容
- 进度条支持点击跳转，拖动时有实时预览
- 倍速切换时平滑过渡，不会跳帧
- 导出过程显示进度条和帧数信息
- 导出完成显示成功提示动画
